# Module 2 – Clock Domain Crossing, Low-Power RTL Design and Advanced SoC Concepts

## Overview

Module 2 focuses on advanced Register Transfer Level (RTL) design techniques used in modern FPGA and ASIC development. The module covers Clock Domain Crossing (CDC), multi-stage synchronizers, asynchronous FIFOs, multi-clock domain architectures, low-power RTL coding, clock gating, reset synchronization, hierarchical design, IP reuse, and reset management techniques.

All designs were implemented using Verilog HDL, simulated using Verilator, verified with GTKWave, and analyzed through waveform inspection to understand real hardware behavior.
